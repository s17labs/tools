# Image Resizer

[Explore the Live Tool »](https://s17labs.github.io/tools/image-resizer)

A privacy-focused, browser-based image manipulation tool. This utility allows you to resize single or multiple images simultaneously without uploading them to a server—all processing happens locally on your device.

## Features

* **Batch Processing**: Upload and resize dozens of images at once.
* **Smart Preview**: A clean gallery view showing up to 6 image previews with a "plus-count" for larger batches.
* **Aspect Ratio Lock**: Maintain image proportions automatically or set custom dimensions in pixels.
* **High Performance**: Generates a .zip file for batch downloads to keep your workflow organized.
* **Midnight Theme**: A sleek, high-contrast UI designed for the modern workspace.

## Technical Details

Built using vanilla HTML5, CSS3, and JavaScript, utilizing:
- **JSZip**: For client-side archiving.
- **Canvas API**: For high-quality image resampling.
- **Aldrich Font**: For that signature s17 Labs tech aesthetic.

## Usage
Simply open the index.html file in any modern browser or visit the [live page](https://s17labs.github.io/tools/image-resizer).
