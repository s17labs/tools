# Tools

A collection of lightweight, high-performance utility tools developed by **s17 Labs**. These tools are designed to run entirely in the browser, ensuring your data remains private and secure while providing professional-grade functionality.

## Available Tools

* **[Image Resizer](https://s17labs.github.io/tools/image-resizer)**: Batch resize images with aspect ratio locking and high-quality output. ([Source](./image-resizer))

## Vision
Our goal is to build a suite of essential utilities for creators and developers that are fast, open-source, and require zero installation.
