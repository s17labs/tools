# Case Converter

[Explore the Live Tool »](https://s17labs.github.io/tools/case-converter)

A fast, secure, and developer-centric text transformation utility. This tool allows you to convert strings between various programming conventions and writing formats instantly—all processing happens locally in your browser to ensure maximum data privacy.

## Features

* **Multi-Format Support**: Instantly transform text into camelCase, snake_case, PascalCase, kebab-case, and many more.
* **Expanded Action Grid**: Access over 15 specialized conversion types through a clean, toggleable interface.
* **Integrated Overlays**: Quick-access buttons for Paste, Copy, and Clear with color-coded visual feedback for successful actions.
* **Smart Tooltips**: Non-intrusive hover tooltips providing clarity on button functions without cluttering the workspace.
* **Lava Design Language**: A high-contrast, dark-mode UI optimized for long-term use in development environments.

## Technical Details

Built using vanilla HTML5, CSS3, and JavaScript, utilizing:
- **Fira Code**: For highly legible, monospace text representation within the editor.
- **RegEx Engine**: For intelligent string parsing and word boundary detection across all case types.
- **Aldrich Font**: For the signature s17 Labs tech aesthetic and branding.

## Usage
Simply open the index.html file in any modern browser or visit the [live page](https://s17labs.github.io/tools/case-converter).
